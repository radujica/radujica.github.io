Book Identifier App

I have implemented this app during my Bachelor Project which aims to bridge the physical--digital barrier of book interactions. More precisely, it provides 5 ways of identifying a book (i.e. finding its ISBN) by using the camera, NFC, and/or visual processing techniques. Moreover, it showcases how these interactions can be used for data analysis and book management in the context of a library.
